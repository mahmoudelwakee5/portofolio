# Premium WordPress Developer Portfolio

A modern, premium portfolio website for Mahmoud Anwar Elwakeel, showcasing WordPress development expertise and project case studies.

## Features

### 🎨 Design
- **Glassmorphism UI**: Modern glass-like design with backdrop blur effects
- **Premium Aesthetics**: Professional color scheme with gradients and shadows
- **Responsive Design**: Mobile-first approach with breakpoints for all devices
- **Smooth Animations**: CSS transitions and JavaScript-powered interactions

### 🚀 Functionality
- **Fixed Navigation**: Sticky navbar with smooth scrolling
- **Mobile Menu**: Hamburger menu for mobile devices
- **Interactive Elements**: Hover effects, floating animations, and reveal animations
- **Contact Form**: Functional email form with validation
- **Project Showcases**: Detailed case studies with live links

### 📱 Sections
1. **Hero**: Compelling introduction with stats and call-to-actions
2. **About**: Personal background and professional highlights
3. **Why Hire Me**: 6 key reasons to choose the developer
4. **Projects**: 5 detailed case studies with results
5. **Services**: Comprehensive WordPress development offerings
6. **Skills**: Technical expertise and proficiencies
7. **Testimonials**: Client feedback and reviews
8. **Contact**: Contact information and inquiry form

## Technologies Used

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **JavaScript**: Interactive features and DOM manipulation
- **Google Fonts**: Inter and Space Grotesk typography
- **Responsive Images**: Optimized for all screen sizes
